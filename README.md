# 🚀 Day 28 – Multi-Agent AI System using AWS Bedrock

## Overview

Today I explored Multi-Agent AI Architecture using AWS Bedrock.

Instead of one AI model handling everything,
multiple specialized agents collaborate together.

---

## What is a Multi-Agent System?

A Multi-Agent System consists of:

- Planner Agent
- Research Agent
- Analysis Agent
- Response Agent

Each agent performs a specific task.
